# Object Oriented Programming With Java

## Overview
This repository contains examples and programs related to **Object-Oriented Programming (OOP) concepts using Java**.  
The purpose of this repository is to help students and beginners understand OOP principles through simple and practical Java programs.

---
 
## OOP Concepts Covered

- Classes and Objects
- Constructors
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Exception Handling
